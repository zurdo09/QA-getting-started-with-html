HTML Basics

As already mentioned in the introduction , the labeling HTML or hypertext language is the language used to create web pages. An html file is simply a text file that can be edited conventional with any basic editor (such as Windows Notepad).

Top Tags

Basic HTML tags, must present throughout the document they are:

  • html : This is the tag that defines the beginning of the HTML document, tells the browser that everything that follows should be treated as a series of html codes.

  • head : Defines the html document header, this header usually contains information about the document not shown directly in the browser. Such as the title of the browser window. Within the header head we can find:

  • title : Defines the title of the page. Usually the title appears in the title bar above the window

  • link : to define some advanced features, such as the style sheets used for page layout, example: link rel = "stylesheet" href = "/ style.css" type = "text / css"

  • body : Defines the main content or document body, this is the part of the HTML document that is displayed in the browser, inside this tag can define properties common to the entire page, as colored background and margins. Pies body can find very many labels. Below some exemplary indicated:

  • h1, h2 ... h6 : headers or titles of the document in different font sizes

  • P : new paragraph

  • br : Forced line break

  • TABLE : start of a table (the rows are identified with tr and the cells within the rows with td)

  • a : it indicates the existence of a hyperlink or link, on or off the website. The last parameter must be defined through the attribute href (example: Google is represented as Google )

  • div : beginning of a special area on the page

  • img : indicates the existence of an image to be displayed in the browser

  • li, ol, ul : One of the most used to manage lists.
